This generates a GUID and puts it into the active file.

## Release Notes

To get `guid-generator.vsix`

```bash
npm install -g vsce
git clone git@github.com:rvehall/GUID-Generator-for-VS-Code.git
cd guid-generator
npm install
vsce package
```
